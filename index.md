## GalaxyLib

1. Add [jitpack](https://jitpack.io/) as a repository.

```markup
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io/</url>
</repository>
```

2. Add the GitHub information

```markup
<dependency>
    <groupId>com.github.imaginedevMC</groupId>
    <artifactId>GalaxyLib</artifactId>
    <version>1.1.0</version>
</dependency>
```
